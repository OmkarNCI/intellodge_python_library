# IntellLodge Core Utilities

Reusable Python modules for CRUD services, logging, validation, API responses, authentication (AWS Cognito), exceptions, and datetime helpers. Designed for Django + DynamoDB projects.

- Install: `pip install -r requirements.txt`
- Activate: `python3 -m venv venv && source venv/bin/activate`
- Usage: `from intellodge_core.base_service import BaseDynamoDBService`
